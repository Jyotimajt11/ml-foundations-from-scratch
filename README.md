# 🤖 ML Foundations From Scratch

This repository documents my hands-on journey of learning Machine Learning from the ground up.

Each project focuses on implementing one machine learning algorithm using Python and Scikit-learn, along with model evaluation, visualization, and detailed explanations of how the algorithm works.

The goal of this repository is to build a strong foundation in machine learning concepts and create a portfolio of practical ML projects.

---

## 🎯 Learning Objectives

Through this repository, I aim to:

- Understand the mathematical intuition behind each algorithm.
- Implement models using real-world datasets.
- Evaluate performance using appropriate metrics.
- Visualize results using plots and charts.
- Compare different machine learning algorithms.
- Build a solid portfolio for internships and job applications.

---

## ✅ Models Completed

### 1. Linear Regression

Predicts continuous numerical values.

**Concepts Learned:**

- Regression analysis
- Best-fit line
- Slope and intercept
- Mean Squared Error (MSE)
- R² Score
- Actual vs Predicted visualization

---

### 2. Multiple Linear Regression

Uses multiple features to predict a continuous target.

**Concepts Learned:**

- Multiple input variables
- Coefficients and weights
- Feature contribution analysis
- Multivariable relationships

---

### 3. Logistic Regression

Used for binary classification problems.

**Concepts Learned:**

- Sigmoid function
- Probability prediction
- Decision boundary
- Accuracy
- Precision
- Recall
- F1-score
- Confusion matrix

---

### 4. K-Nearest Neighbors (KNN)

Classifies data points based on nearest neighbors.

**Concepts Learned:**

- Distance-based learning
- Euclidean distance
- Choosing the optimal K
- Feature scaling
- Accuracy comparison

---

### 5. Decision Tree

Builds interpretable tree-based decision rules.

**Concepts Learned:**

- Root node
- Decision node
- Leaf node
- Gini impurity
- Information gain
- Tree depth
- Feature importance
- Decision tree visualization

---

### 6. Random Forest

An ensemble method that combines multiple decision trees for improved accuracy and reduced overfitting.

**Concepts Learned:**

- Ensemble learning
- Bootstrap sampling
- Random feature selection
- Majority voting
- Out-of-bag (OOB) estimation
- Feature importance
- Confusion matrix analysis

## 🚀 How to Run Any Project

1. Clone the repository:

```bash
git clone https://github.com/jyotima-tomar/ml-foundations-from-scratch.git
```

2. Navigate to the repository:

```bash
cd ml-foundations-from-scratch
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run any model:

```bash
py Random_Forest/src/random_forest_model.py
```

Replace the path above with the project you want to run.
