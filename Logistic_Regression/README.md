# Logistic Regression Classification Model

## Overview

This project implements Logistic Regression as part of my Machine Learning Foundations journey.

Logistic Regression is used for classification problems where the output belongs to categories.

Example:

- 0 = No
- 1 = Yes

## Model Type

Supervised Learning - Classification

## Concept

Logistic Regression predicts probability and converts it into a class label.

It uses the sigmoid function to convert values between 0 and 1.

## Workflow

1. Load dataset
2. Separate features and target
3. Split data into training and testing sets
4. Train Logistic Regression model
5. Make predictions
6. Evaluate using accuracy and confusion matrix
7. Visualize model results

## Evaluation Metrics

- Accuracy
- Confusion Matrix
- Classification Report
- Precision
- Recall
- F1-score
